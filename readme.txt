Integrate FormAgent.ai chat widget into your WordPress website with just one Agent ID.

== Description ==

FormAgent.ai Integration is a simple yet powerful WordPress plugin that allows you to easily embed the FormAgent.ai chat widget on your website. With just your Agent ID, you can provide AI-powered customer support and engage with your visitors.

**How it works:**

5. The chat widget will automatically appear on all your website pages

== Installation ==

**Automatic Installation:**

1. Login to your WordPress admin panel
2. Go to Plugins > Add New
3. Search for "FormAgent.ai Integration"
4. Click "Install Now" and then "Activate"

**Manual Installation:**

1. Download the plugin zip file
2. Login to your WordPress admin panel
3. Go to Plugins > Add New > Upload Plugin
4. Choose the downloaded zip file and click "Install Now"
5. Activate the plugin

**Setup:**

1. After activation, go to Settings > FormAgent.ai
2. Enter your Agent ID (get it from your FormAgent.ai dashboard)
3. Click "Save Settings"
4. Visit your website to see the chat widget in action

== Frequently Asked Questions ==

= Where do I get my Agent ID? =

You can get your Agent ID from your FormAgent.ai dashboard at https://formagent.ai. If you don't have an account yet, you'll need to sign up first.

= Will this slow down my website? =

No, the plugin is designed to be lightweight. The chat widget script is loaded asynchronously and won't affect your page load times.

= Can I customize the chat widget appearance? =

The widget appearance is controlled through your FormAgent.ai dashboard settings. You can customize colors, position, and behavior from there.

= Does this work with any WordPress theme? =

Yes, the plugin works with any properly coded WordPress theme. The chat widget is injected into the footer area of your website.

= Is this plugin free? =

The WordPress plugin itself is completely free. However, FormAgent.ai is a paid service with various pricing plans available.

= What languages are supported? =

The plugin interface supports English, Chinese (Simplified), and Japanese. The chat widget language depends on your FormAgent.ai configuration.
